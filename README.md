# 🎓 College Management System

## 📌 Project Overview

The **College Management System** is a web-based application designed to simplify and automate the management of various college activities such as student records, faculty details, course information, and administrative tasks.

This system helps educational institutions maintain organized data and improves efficiency by reducing manual paperwork.

---

## 🚀 Features

* 👨‍🎓 Student Registration and Management
* 👩‍🏫 Faculty Information Management
* 📚 Course and Subject Management
* 📝 Student Admission Form
* 📊 Display of Student and Faculty Records
* 🔐 User-friendly Interface for Admin

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/college-management-system.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd college-management-system
```

### 3️⃣ Install Required Packages

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Server

```bash
python manage.py runserver
```

### 5️⃣ Open in Browser

```
http://127.0.0.1:8000/
```
